# Slack-clone application

## Features
- Register with email/password or Github/Google auth
- Create Workspaces/Channels
- Invite others to Workspace
- Public chat
- Direct messages
- Support therads and image upload

## Technology

### Backend

- Next.js
- Convex auth
- Convex database
- Convex storage

### Frontend

- React TypeScript
- Tailwind CSS
- Shadcn UI
